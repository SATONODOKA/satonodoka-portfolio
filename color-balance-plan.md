# カラーバランス改善プラン

## 現状分析

### 現在のカラー使用状況
- **白背景**: Hero、HowIWork、全体のbody
- **薄いグレー背景 (bg-gray-50)**: FeaturedProjects、Contact
- **白いカード**: すべてのCardコンポーネント
- **アクセントカラー (青 #2563eb)**: ボタン、リンクのみ

### 問題点
- 白が多すぎて単調
- セクション間の区切りが不明確
- カードが背景に埋もれている
- 視覚的な階層が弱い

## 改善プラン

### 1. セクション背景の交互配置
- **Hero**: 白背景（現状維持）
- **FeaturedProjects**: 薄いグレー (bg-gray-50) → **やや濃いグレー (bg-gray-100)** に変更
- **HowIWork**: 白背景 → **薄いグレー (bg-gray-50)** に変更
- **Contact**: 薄いグレー (bg-gray-50) → **白背景** に変更

### 2. カードの背景色調整
- **Cardコンポーネント**: 白背景 → **薄いグレー (bg-gray-50)** に変更
- ホバー時: より濃いグレー (bg-gray-100) に変更
- ボーダー: より濃いグレー (border-gray-200) に変更

### 3. ヘッダー・フッターの背景
- **Header**: 白背景 → **薄いグレー (bg-gray-50)** に変更
- **Footer**: 白背景 → **薄いグレー (bg-gray-50)** に変更

### 4. テキストの階層強化
- **見出し**: 現状維持（濃いグレー）
- **本文**: グレーの濃淡で階層を明確化
- **メタ情報**: より薄いグレーで控えめに

### 5. ボーダーとシャドウの活用
- セクション間のボーダーを追加
- カードのシャドウを少し強化
- ヘッダーのボーダーを維持

## 実装順序

1. **セクション背景の調整**（Hero、FeaturedProjects、HowIWork、Contact）
2. **Cardコンポーネントの背景色変更**
3. **Header・Footerの背景色変更**
4. **テキスト階層の微調整**
5. **ボーダー・シャドウの調整**

## 期待される効果

- 白が多すぎる問題の解消
- セクション間の区切りが明確に
- カードが背景から浮き上がって見える
- 視覚的な階層が強化される
- 白黒青の色は維持しつつ、バランスが改善
