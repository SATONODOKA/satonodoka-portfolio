# セクションバランス調整プラン

## 現状の問題点

1. **セクション間の高さが不統一**
   - Hero: コンテンツ量が少ない
   - Projects: 3つのプロジェクトカードで高さが長い
   - How I Work: ステップ3つ + 技術スタック/AIツールで高さが非常に長い
   - Contact: コンテンツ量が少ない

2. **視覚的なリズムの欠如**
   - セクション間の余白は統一されているが、コンテンツ量の差が大きすぎる
   - 背景色の違い（白、グレー100、グレー50）が視覚的な不統一感を生んでいる

3. **スクロール体験の不自然さ**
   - 短いセクションと長いセクションが交互に来ることで、リズムが崩れている

## 解決策

### 1. セクションの最小高さを設定
- 各セクションに最小高さ（min-height）を設定
- ビューポートの高さを基準に、統一感のある高さを設定

### 2. コンテンツの配置を最適化
- Hero: コンテンツを中央揃えにして、視覚的なバランスを改善
- Projects: カード間の余白を調整
- How I Work: ステップと技術スタックの配置を最適化
- Contact: コンテンツを中央揃えにして、高さを確保

### 3. 背景色の統一
- 交互に背景色を配置（白 → グレー → 白 → グレー）
- または、すべて白背景にして、ボーダーで区切る

### 4. セクション間の余白を調整
- セクション間の余白を統一
- コンテンツ量に応じて、内部の余白を調整

## 実装方針

1. **Heroセクション**
   - 最小高さ: 80vh（ビューポートの80%）
   - コンテンツを垂直方向に中央揃え

2. **Projectsセクション**
   - 最小高さ: 60vh
   - カード間の余白を調整

3. **How I Workセクション**
   - 最小高さ: 70vh
   - ステップと技術スタックの配置を最適化

4. **Contactセクション**
   - 最小高さ: 50vh
   - コンテンツを中央揃え
